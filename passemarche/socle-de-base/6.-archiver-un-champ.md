# 6. Archiver un champ

Lorsqu’un champ ne doit plus être utilisé, il peut être **archivé**.

L’archivage permet de retirer un champ de l’usage courant sans le supprimer définitivement de la base.

Un champ archivé :

* n’apparaît plus dans la liste active du socle de base,
* n’est plus proposé dans les nouveaux usages ou formulaires,
* mais reste conservé dans le système.

L’archivage n’efface pas les données déjà associées à ce champ.\
Les informations existantes restent donc disponibles dans les dossiers ou formulaires où ce champ a déjà été utilisé.

Cette logique permet de préserver la cohérence des données historiques tout en retirant le champ des usages futurs.\
