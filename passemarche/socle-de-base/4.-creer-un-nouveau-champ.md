# 4. Créer un nouveau champ

La création d’un champ permet d’enrichir le socle de base avec une nouvelle information à collecter ou à afficher dans Passe Marché.

Cette action est réservée aux utilisateurs disposant des droits appropriés.

#### Lancer la création

Depuis l’écran **Socle de base**, l’utilisateur autorisé peut initier la création d’un nouveau champ via l’action dédiée.\
<br>

#### Définir les informations générales

Lors de la création, il convient de renseigner les caractéristiques essentielles du champ, notamment :

* le **type de champ**,
* son éventuel caractère **obligatoire**,
*   le ou les **types de marché** auxquels il s’applique.\
    \
    Le champ peut ensuite être configuré selon deux modes :

    * **récupération automatique via API**,
    * ou **saisie manuelle**.

    Lorsque le mode API est retenu, il est nécessaire de préciser la source de données ainsi que l’information attendue.



Le champ doit également être décrit séparément pour :

* la **vue acheteur**,
* la **vue candidat**.

Pour chaque vue, l’administrateur renseigne :

* la catégorie,
* la sous-catégorie,
* le titre,
* et, si besoin, une description.

Une fois l’ensemble des informations requises complétées, la création peut être validée.\
Un bouton d’annulation permet à tout moment d’abandonner l’opération sans enregistrer les données saisies.

Après validation, le nouveau champ est ajouté à la liste du socle de base à l’emplacement correspondant à la catégorie et à la sous-catégorie choisies.<br>
