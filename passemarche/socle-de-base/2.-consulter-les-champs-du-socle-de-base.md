# 2. Consulter les champs du socle de base

L’écran **Socle de base** permet de visualiser l’ensemble des champs actifs du référentiel.

Chaque champ est présenté dans une liste structurée, avec les éléments utiles à sa compréhension, notamment :

* sa catégorie et sa sous-catégorie,
* son intitulé,
* son type,
* sa source,
* ainsi que les types de marché auxquels il s’applique.

Depuis cette vue, un administrateur ou un utilisateur autorisé peut consulter le détail d’un champ pour en comprendre la configuration ou, selon ses droits, le modifier.\
