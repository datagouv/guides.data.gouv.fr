# 7. Télécharger et analyser le dossier de candidature ZIP

Pour une analyse plus approfondie, vous pouvez télécharger un **dossier de candidature au format ZIP**.

Ce dossier contient :

* les documents transmis par le candidat,
* les documents récupérés automatiquement,
* une **attestation récapitulative à destination de l’acheteur**.

Les documents sont conservés dans leur **format d’origine**, afin de garantir leur lisibilité et leur intégrité.

_Pour les marchés avec des lots de types différents, les documents sont organisés par sous-dossiers._ [_Voir la gestion des lots._](9.-gestion-des-lots.md)

#### &#x20; Organisation du dossier

Lorsque le marché comporte des lots de plusieurs types, le dossier ZIP est **organisé en sous-dossiers** pour faciliter votre analyse :

* un dossier **« Commun »**, regroupant les pièces administratives demandées une seule fois, quelle que soit la combinaison de lots,
* un dossier par **type de lot** auquel le candidat postule : **Travaux**, **Services** ou **Fournitures**, contenant les pièces spécifiques à ce type.

Seuls les dossiers correspondant aux types réellement présents dans la candidature sont créés : vous ne trouverez ni dossier vide, ni pièce dupliquée, une pièce commune à tous les types figure uniquement dans le dossier « Commun ».
