# 4. Vérifier et valider la configuration du marché

Avant l’ouverture aux candidatures, Passe Marché vous propose une **vue de synthèse**.

Cette synthèse vous permet de vérifier :

* les éléments obligatoires qui seront demandés à tous les candidats,
* les éléments complémentaires que vous avez choisi d’ajouter, le cas échéant.

Tant que la validation finale n’est pas effectuée, vous pouvez revenir aux étapes précédentes pour ajuster la configuration.

{% hint style="info" %}
Lorsque vous cliquez sur **« Finaliser la configuration »** :

* la configuration est enregistrée et transmise à votre plateforme de marchés publics,
* le formulaire de candidature devient accessible aux candidats.

La configuration reste **modifiable tant que la consultation n'est pas publiée** sur votre plateforme. C'est la **publication de la consultation** qui verrouille définitivement la configuration.
{% endhint %}
