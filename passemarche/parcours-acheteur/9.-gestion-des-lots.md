# 9. Gestion des lots

### Objectif

Permettre à l’acheteur de configurer un marché comportant plusieurs lots afin de :

* structurer son marché ;
* configurer les éléments concernant les lots du marché
*   permettre aux candidats de candidater à un ou plusieurs lots.\
    \
    Une fois les lots créés :

    * chaque lot correspond à une partie du marché
    * les entreprises pourront choisir librement les lots auxquels elles souhaitent répondre

    Vous n’imposez jamais à une entreprise de répondre à un lot précis.\

### Réception des lots depuis votre profil acheteur

Vous arrivez dans Passe Marché **depuis votre plateforme d'achat** (PLACE, AWS, Atexo…). Les données du marché — dont la liste des lots — sont transmises automatiquement par votre profil acheteur. Vous n'avez rien à ressaisir : votre travail consiste à **vérifier, corriger si besoin, puis configurer**.



👉 **Bon à savoir.** Chaque lot porte un **type** : **Travaux**, **Services** ou **Fournitures**. C'est ce type qui détermine les pièces justificatives demandées aux candidats. C'est donc l'information à vérifier en priorité.

### Étape 1 — Vérifier les types de lots reçus

Passe Marché affiche la liste de tous vos lots avec, pour chacun, le **type transmis par votre plateforme**. Un bandeau d'information vous indique explicitement **quelle plateforme** a transmis ces types.

Selon ce que votre éditeur sait envoyer, deux situations :

* **Le type est fourni par lot** — chaque lot affiche déjà son type. Vous n'avez qu'à vérifier.
* **Un seul type est fourni pour tout le marché** — Passe Marché applique ce type à tous les lots. À vous de confirmer qu'il est correct, ou de l'ajuster lot par lot.

👉 Prenez le temps de contrôler : **un mauvais type = de mauvaises pièces demandées** au candidat.

### Étape 2 — Corriger le type d'un ou plusieurs lots

Si un type est inexact, vous pouvez le modifier — sur un seul lot ou sur plusieurs d'un coup.

1. Cliquez sur **« Modifier les types de lot »**. Des cases à cocher apparaissent en face de chaque lot.
2. Cochez le ou les lots concernés (un lien **« Tout cocher / Tout décocher »** est disponible, pratique pour les marchés à plus de 10 lots).
3. Un bandeau d'action s'affiche : cliquez sur **« Modifier le type »**.
4. Choisissez le type — **Travaux**, **Services** ou **Fournitures** — puis **« Appliquer »**.

Le type est appliqué **à tous les lots cochés en une seule action**.



### Étape 3 — Limiter le nombre de lots (facultatif)

Vous pouvez fixer un **nombre maximum de lots** auxquels une même entreprise pourra candidater. Cette limite est ensuite transmise au formulaire candidat et appliquée automatiquement.

Ce paramètre est **indépendant** du type des lots : modifier un type ne change rien à la limite, et inversement.

#### Exemple

* Votre marché contient 5 lots
* Vous fixez une limite à 2
* Une entreprise pourra candidater à 1 ou 2 lots, mais pas plus

### Étape 4 — Configurer les pièces justificatives

C'est ici que Passe Marché vous fait gagner du temps. Le parcours dépend de la composition de votre marché.

#### Cas A — Tous les lots ont le même type

Rien de nouveau : vous configurez les pièces **comme pour un marché classique**, en fonction de ce type unique. Vous pouvez directement passer à [la configuration des exigences](3.-parametrer-les-exigences-de-candidature.md) (pièces demandées, informations, etc.)&#x20;

#### Cas B — Le marché mélange plusieurs types (marché hétérogène)

Vous configurez les pièces **une seule fois**, pas pour chaque lot.

**Une seule liste de pièces, pas un formulaire par lot.** Au lieu de répéter la configuration pour chaque lot, Passe Marché vous présente **une seule carte « Formulaire de candidature »**. Elle porte les badges des types présents dans le marché — par exemple _Travaux · Services._ Vous cliquez sur **« Configurer »** pour ouvrir la liste des pièces.

&#x20;En face de chaque pièce demandée, vous lui attribuez une **portée** au moyen d'un badge :

* **« Commun aux lots »** → la pièce est demandée à **tous les candidats**, quel que soit le lot visé. Ce sont typiquement les pièces administratives :  attestations fiscales et sociales, etc.
* **Type spécifique** — **Travaux**, **Services** ou **Fournitures** → la pièce n'est demandée **qu'aux candidats qui postulent à un lot de ce type**. Par exemple, une référence de chantier sur un badge « Travaux » ne sera jamais réclamée à un candidat qui ne vise que des lots de Services.

**Le résultat : un formulaire sur-mesure pour chaque candidat.** À partir de ces étiquettes, Passe Marché **assemble automatiquement le formulaire** de chaque entreprise selon les lots qu'elle a choisis. Un candidat ne voit que les pièces qui le concernent vraiment : les pièces communes, plus celles correspondant aux types de ses lots. Vous configurez **une fois** ; le système gère les combinaisons.

### Étape 5 — Transmettre la configuration

Une fois le formulaire validé, le bouton **« Transmettre la configuration »** s'active. Un clic, et la configuration repart vers votre plateforme d'achat. Les candidats peuvent désormais déposer leur dossier.
