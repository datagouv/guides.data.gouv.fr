# 5. Finaliser la configuration et revenir sur votre plateforme

Après validation, Passe Marché procède à la transmission des informations vers votre plateforme de marchés publics.

Une page intermédiaire s’affiche pour vous informer que la transmission est en cours :

* en cas de succès, un message de confirmation apparaît, suivi d’une **redirection automatique** vers votre plateforme,
* en cas d’échec, un message explicite vous informe de la situation et vous propose de réessayer ou de contacter l’assistance.

![](<../.gitbook/assets/image (2) (1)>)



### Modifier la configuration avant la publication

Tant que la consultation n'est pas publiée sur votre plateforme de marchés publics, vous pouvez **modifier la configuration de votre marché**.

Depuis votre plateforme, vous êtes redirigé vers Passe Marché avec la **configuration actuelle pré-chargée** :

* vous ajustez les exigences de candidature comme lors de la configuration initiale,
* après validation, la **nouvelle configuration remplace la précédente** et est transmise à votre plateforme.

{% hint style="warning" %}
Dès que la consultation est **publiée** sur votre plateforme, la configuration est **définitivement verrouillée**. Toute tentative de modification après publication affiche une page d'information vous indiquant que la modification n'est plus possible.
{% endhint %}
