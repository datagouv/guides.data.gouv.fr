# 13. Gestion des lots

### Choisir les lots auxquels vous souhaitez répondre

Lorsque vous candidatez à un marché, vous pouvez être amené à choisir un ou plusieurs lots.

Ce que vous voyez

Une liste de lots à sélectionner :

<figure><img src="../.gitbook/assets/image (8).png" alt=""><figcaption></figcaption></figure>

### Sélectionner vos lots

Vous pouvez :

* choisir un seul lot
* sélectionner plusieurs lots
*   ou tous les lots (si aucune limite n’est définie)\
    Parfois, l’acheteur **limite le nombre de lots** auxquels vous pouvez répondre.

    Exemple

    > Vous pouvez sélectionner jusqu’à 2 lots maximum

    Dans ce cas :

    * une fois la limite atteinte, **vous ne pouvez plus** sélectionner d’autres lots
    * les autres lots apparaissent grisés

    Si vous souhaitez changer :

    il suffit de décocher un lot pour en sélectionner un autre

Il n’y a aucune obligation de répondre à tous les lots.

<figure><img src="../.gitbook/assets/image (9).png" alt=""><figcaption></figcaption></figure>

### Remplir votre candidature

Une fois vos lots sélectionnés :

* vous accédez au formulaire de candidature
* vos informations sont partagées entre tous les lots

👉 Vous ne remplissez qu’un seul formulaire, même pour plusieurs lots

<figure><img src="../.gitbook/assets/image (10).png" alt=""><figcaption></figcaption></figure>

Une carte apparaît pour vous indiquer votre progression.

Elle affiche :

* le nombre de lots sélectionnés
* votre niveau de complétion (ex : 20/30 champs)
* un accès au formulaire

### Compléter votre dossier

Vous pouvez :

* remplir votre dossier progressivement
* revenir plus tard pour compléter ou modifier

👉 Vos informations sont enregistrées automatiquement<br>

<figure><img src="../.gitbook/assets/image (11).png" alt=""><figcaption></figcaption></figure>

<figure><img src="../.gitbook/assets/image (12).png" alt=""><figcaption></figcaption></figure>

### Envoyer votre candidature

Vous pouvez envoyer votre candidature :

* même si le formulaire n’est pas entièrement complété

👉 Tout est transmis en une seule fois, pour tous les lots sélectionnés

### Vérifier votre attestation

Après envoi, votre attestation récapitule :

* les lots auxquels vous avez candidaté
* leurs informations principales

👉 Seuls les lots que vous avez sélectionnés apparaissent
