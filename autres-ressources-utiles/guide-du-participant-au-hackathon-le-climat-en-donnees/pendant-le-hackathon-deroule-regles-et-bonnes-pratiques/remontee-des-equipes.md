# Remontée des équipes

{% hint style="success" %}
**Faites-nous remonter votre équipe au plus tard le 2 décembre, à 19h45, en remplissant** [**ce formulaire**](https://grist.numerique.gouv.fr/o/datagouv/forms/nTkP3UdFxSBkTVzKGmj8RR/13)**.**
{% endhint %}

Au terme de la 1ère journée de hackathon, nous vous invitons à nous faire remonter votre équipe et une description succincte de votre projet.

Cela nous permettra de vous identifier et d’organiser la présentation devant les mentors et la présélection des projets.

Nous vous demanderons de nous préciser les éléments suivants :

| Nom de l’équipe/du projet                   |   |
| ------------------------------------------- | - |
| Membres de l’équipe et leurs compétences    |   |
| Défi(s) choisi(s)                           |   |
| Objectif du projet et problématique résolue |   |
| Approche adoptée                            |   |
| Usages pressentis et bénéfices              |   |

Cette description ne vous engage en rien, et vous pourrez toujours changer votre approche.
