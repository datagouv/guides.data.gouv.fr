# Soumission du projet

{% hint style="success" %}
**Vous devrez soumettre votre projet le mercredi 3 décembre, avant 17h00.**
{% endhint %}

**Pour soumettre votre projet, publiez-le comme réutilisation sur** [**data.gouv.fr**](http://data.gouv.fr)**.**

La marche à suivre est détaillée [ici](https://guides.data.gouv.fr/guide-data.gouv.fr/reutilisations/publier-une-reutilisation).

Vous pourrez toujours modifier votre réutilisation à la suite de l’événement.

{% hint style="success" %}
**La liste de mots clés devra intégrer le mot-clé “hackathon-climat” (pour nous permettre d’identifier votre projet plus facilement).**
{% endhint %}

La description de la réutilisation publiée devra respecter le format suivant :

| Partie                                            | Description                                                                                                                                                                                                                                                                                                                                                                                                                               |
| ------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Description générale du projet**                |                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| **Problématique et proposition de valeur**        | <ul><li>A quelle problématique s’attaque le projet ?</li><li>Quelle est la proposition de valeur ?</li></ul>                                                                                                                                                                                                                                                                                                                              |
| **Solution**                                      | <ul><li>Description de la solution et de ses fonctionnalités</li><li>Quel usage est fait des données ? Que vous permettent-elles de faire ?</li><li>Quelle est la méthode de création de la solution ?</li></ul>                                                                                                                                                                                                                          |
| **Impact envisagé**                               | <ul><li>Que permet de faire la solution ?</li><li>Qui sont les usagers visés, quels bénéfices ?</li></ul>                                                                                                                                                                                                                                                                                                                                 |
| **Ressources**                                    | <ul><li>Livrables</li><li>Dépôt de code</li></ul>                                                                                                                                                                                                                                                                                                                                                                                         |
| **Retours consolidés sur les données exploitées** | <ul><li>Quelles erreurs, incohérences ou anomalies identifiées ? (techniques et scientifiques)</li><li>Quelles difficultés d’exploitation ? Quelles pistes de résolution ?</li><li>Quelles limites pour le cas d’usage traité ? Les données proposées sont-elles adaptées aux exigences de votre domaine sectoriel ou géographique d’application (résolution spatiale et temporelle, couverture spatiale et temporelle, etc.) ?</li></ul> |

\[OPTIONNEL] Vos productions annexes (jeux de données retravaillés, etc.) pourront également être publiés sur [data.gouv.fr](http://data.gouv.fr/) en tant que ressource communautaire. La marche à suivre est détaillée [ici](https://guides.data.gouv.fr/guide-data.gouv.fr/ressource-communautaire).
