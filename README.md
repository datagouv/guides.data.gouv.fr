# Guide DataGouv

Ce dépôt contient les guides lié à l'écosystème **DataGouv**.

## Synchronisation avec GitBook

Ce repository GitHub est **synchronisé avec le GitBook** du guide.  
Les modifications peuvent donc être effectuées de deux manières :

### 1. Via GitHub
- Proposer vos modifications via une **Pull Request**

### 2. Via GitBook
- Modifier directement le contenu depuis **l’interface GitBook**

Les changements effectués sur GitBook sont automatiquement synchronisés avec ce dépôt.
