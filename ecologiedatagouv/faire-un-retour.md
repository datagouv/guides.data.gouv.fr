# Faire un retour

Vos retours sont les bienvenus pour nous permettre d'améliorer le catalogue.&#x20;

Si votre question porte sur le fonctionnement du catalogue, vous pouvez écrire à l’équipe [_ecologie_.**data.gouv**._fr_](mailto:ecospheres@developpement-durable.gouv.fr). Si elle porte plus spécifiquement sur les indicateurs territoriaux, nous vous invitons à écrire à l’équipe [indicateurs](mailto:indicateurs-transition-ecologique@developpement-durable.gouv.fr).

Si votre question concerne un jeu de données, un indicateur ou un bouquet, sollicitez l'aide de la communauté via **l'onglet "Discussions"** de la page correspondante en vous connectant.&#x20;
