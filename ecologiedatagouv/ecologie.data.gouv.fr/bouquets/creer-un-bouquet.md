# Créer un bouquet

### Qui peut publier un bouquet ?

Face à l’urgence, la planification écologique mobilise l’ensemble des acteurs – État, collectivités, entreprises, associations, citoyens – pour bâtir une action cohérente et efficace vis-à-vis des défis contemporains.&#x20;

{% hint style="success" %}
**Vous disposez d'une expertise ?** Partagez-la à travers de la publication de vos bouquets.&#x20;
{% endhint %}

{% hint style="warning" %}
Pour être publiés, vos bouquets doivent :&#x20;

* Être rattachés à une organisation ;
* Être validés par un membre de l'équipe chargé d'[_ecologie_.**data.gouv**._fr_](https://ecologie.data.gouv.fr) en contactant l'équipe[ecospheres@developpement-durable.gouv.fr](mailto:ecosphreres@developpement-durable.gouv.fr).
{% endhint %}

{% hint style="info" %}
Ils peuvent également demeurer privés pour votre usage personnel.&#x20;
{% endhint %}

### Comment publier un bouquet ?

{% stepper %}
{% step %}
#### **Rendez-vous sur :** [_ecologie._**data.gouv.**_fr_](http://ecologie.data.gouv.fr), puis cliquer sur l’onglet “bouquet”
{% endstep %}

{% step %}
#### Sélectionner "ajouter un bouquet"

<figure><img src="../../.gitbook/assets/Dark mode gitbook EDG(3) (1).png" alt="" width="188"><figcaption></figcaption></figure>
{% endstep %}

{% step %}
#### Décrire le bouquet&#x20;

Renseignez le sujet, l’objectif, la thématique et la couverture territoriale de votre bouquet.&#x20;

<figure><img src="../../.gitbook/assets/image (7).png" alt=""><figcaption></figcaption></figure>

Dans l'encadré dédié à l'objectif de votre bouquet, il est recommandé de suivre le modèle suivant afin de favoriser la compréhension et la réutilisation du bouquet par d'autres utilisateurs.

{% hint style="success" %}
1. **Présentation**\
   &#xNAN;_&#x44;ans quels enjeux, objectifs ou politique publique s'inscrit ce bouquet ? Quelles décisions l'initiative associée au bouquet cherche-t-elle à éclairer ? Quelles sont les solutions proposées ?_
2. **Public cible**\
   &#xNAN;_&#xC0; qui cette initiative s'adresse-t-elle ?_
3. **Porteurs du projet**\
   &#xNAN;_&#x4D;entionner l'ensemble des partenaires._
4. **Liens utiles**\
   &#xNAN;_&#x4C;ien vers tout portail, application, documentation, code sources et autres supports pertinents._
5. **Point de contact**
{% endhint %}

Vous devez choisir de publier le bouquet en votre propre nom ou en tant qu'organisation.

{% hint style="info" %}
Si vous vous inscrivez dans le cadre d’une organisation, tous les membres de l'organisation pourront éditer le bouquet.
{% endhint %}

Après avoir complété ces étapes, vous pouvez enregistrer votre bouquet. Votre bouquet est désormais créé.&#x20;

{% hint style="info" %}
Le bouquet sera alors créé en mode brouillon et visible uniquement par les membres de l'organisation.&#x20;
{% endhint %}
{% endstep %}

{% step %}
#### Ajouter les jeux de données et indicateurs qui composent le bouquet

Pour ce faire, vous devez cliquer sur "ajouter un jeu de données" lorsque vous vous trouvez sur la page de votre bouquet et intégrer vos jeux de données.

<figure><img src="../../.gitbook/assets/image (8).png" alt=""><figcaption></figcaption></figure>

Renseignez le libellé du jeu de données et sa raison d'utilisation. Vous pouvez ensuite rechercher un jeu de donnée dans [**data.gouv**._fr_](https://data.gouv.fr) ou ajouter l'URL du jeu de données.&#x20;

Toute donnée ou indicateur présent sur [**data.gouv.**_fr_](https://data.gouv.fr) peut être associé à un bouquet.

{% hint style="info" %}
[_ecologie._**data.gouv.**_fr_](../../) et [**data.gouv.**_fr_](https://data.gouv.fr) partagent **le même catalogue** de données. Cela vous permet de récupérer les données pertinentes pour votre cas d'usages parmi toutes celles disponibles sur [**data.gouv.**_fr_](https://data.gouv.fr).&#x20;
{% endhint %}

Une fonctionnalité de regroupement est disponible pour rassembler certains jeux de données par thématique.&#x20;

<figure><img src="../../.gitbook/assets/image (15).png" alt="" width="527"><figcaption></figcaption></figure>

<figure><img src="../../.gitbook/assets/image (16).png" alt="" width="529"><figcaption></figcaption></figure>

{% hint style="info" %}
Si vous disposez de jeux de données pertinents pour votre bouquet et non disponibles sur [**data.gouv**._fr_](https://data.gouv.fr)_,_ il est recommandé de publier vos jeux de données sur [**data.gouv**._fr_](https://data.gouv.fr) en suivant [ces étapes](https://app.gitbook.com/o/w6D6SnLwCXQaMMSzcTvp/s/K56ETHxhBea8DHmbPCgt/~/changes/3/ecologie.data.gouv.fr/donnees/publier-des-donnees), puis de les ajouter à votre bouquet.&#x20;
{% endhint %}
{% endstep %}

#### Modifier le bouquet après sa publication&#x20;

Il est toujours possible de modifier un bouquet dont vous êtes propriétaire après sa publication. Il suffit de vous connecter sur [_ecologie_.**data.gouv**._fr_](https://ecologie.data.gouv.fr) avec le compte propriétaire du bouquet ou un compte inscrit dans l'organisation associée et d'éditer le bouquet.

<figure><img src="../../.gitbook/assets/Dark mode gitbook EDG(2) (6).png" alt="" width="188"><figcaption></figcaption></figure>

{% hint style="info" %}
Tous les membres de l'organisation peuvent éditer un bouquet. [Découvrir comment rejoindre une organisation](https://guides.data.gouv.fr/guide-data.gouv.fr/creer-un-compte-utilisateur-et-rejoindre-une-organisation).&#x20;
{% endhint %}
