# Rechercher un bouquet

Pour rechercher un bouquet, dirigez vous vers l'onglet [Bouquets](./). Outre la barre de recherche générale, où vous pouvez explorer la base de bouquets par mots-clés, il est proposé de filtrer votre recherche par :

1. **Thématique ;**
2. **Couverture territoriale.**

### Thématique

Pour atteindre nos objectifs nationaux et territoriaux, le Secrétariat Général de la planification écologique a défini des leviers d'actions concernant 6 thématiques clés dans le cadre du plan **France Nation Verte** :&#x20;

1. Mieux se loger ;
2. Mieux se nourrir ;
3. Mieux se déplacer ;&#x20;
4. Mieux consommer ;&#x20;
5. Mieux produire ;
6. Mieux préserver et valoriser nos écosystèmes.

Les bouquets peuvent être référencés par ces 6 thématiques FNV.&#x20;

Pour en savoir plus sur le plan France Nation Verte :&#x20;

{% embed url="https://www.info.gouv.fr/grand-dossier/france-nation-verte" %}

### Couverture territoriale

Les bouquets sont défini sur un territoire cible. Ils représentent des initiatives [nationales ](https://ecologie.data.gouv.fr/bouquets/indicateurs-de-planification-ecologique-regionale-cops)ou [territoriales](https://ecologie.data.gouv.fr/bouquets/iarbre-enraciner-la-resilience-climatique-des-territoires-en-cultivant-les-donnees-1).&#x20;
{% endstep %}
{% endstepper %}

Chaque bouquet détaille la présentation du cas d'usage et permet d'accéder aux données et indicateurs qu'il regroupe.&#x20;
