# Tests de qualité

Afin d'assurer la qualité des indicateurs que nous publions, nous faisons tourner un ensemble de tests sur nos tables d'indicateurs, qui nous aident à identifier des erreurs dans les données.

### Test d'unicité

Nous faisons tourner sur chaque table d'indicateur des scripts qui nous permettent d'identifier des doublons dans nos données afin de traiter ces erreurs avant publication des indicateurs.&#x20;

### Test de non nullité

Nous faisons tourner sur chaque table d'indicateur des scripts qui nous permettent d'identifier des valeurs nulles dans nos données afin de traiter ces erreurs avant publication des indicateurs
