# Publication d'un indicateur

Les indicateurs mis à disposition sur [_ecologie._**data.gouv**._fr_](https://ecologie.data.gouv.fr) résultent de méthodologies de collecte des données, de traitement et un travail sur les métadonnées important.

Nous tâchons dans cette partie d'apporter de la transparence sur nos méthodes de travail pour mettre à disposition des indicateurs.

{% hint style="info" %}
Pour en savoir plus sur notre méthodologie de sélection des indicateurs, rendez vous sur cette [page dédiée](methodologie-de-selection-des-indicateurs.md).&#x20;
{% endhint %}
