# Sélection des données

{% hint style="success" %}
Tout jeu de données utile à l’élaboration, à la mise en œuvre ou à l’évaluation des politiques publiques du pôle ministériel chargé de l’aménagement du territoire et de la transition écologique est éligible au référencement sur [_ecologie_**.data.gouv**_.fr_](https://ecologie.data.gouv.frhttps/ecologie.data.gouv.fr/)_._&#x20;
{% endhint %}

[_ecologie_**.data.gouv**_.fr_](https://ecologie.data.gouv.frhttps/ecologie.data.gouv.fr/) s'appuie sur la base de données de [**data.gouv**._fr_](https://data.gouv.fr/)_._ La sélection des jeux de données exposés depuis [_ecologie_**.data.gouv**_.fr_](https://ecologie.data.gouv.frhttps/ecologie.data.gouv.fr/) est opérée au niveau des [organisations](https://www.data.gouv.fr/organizations/) publiant sur [**data.gouv**._fr_](https://data.gouv.fr/)_,_ et non sur une sélection individuelle des jeux de données.&#x20;

Ce mode de sélection limite, pour l’heure, le référencement aux acteurs relevant du pôle ministériel ou intervenant de manière prépondérante sur l’une de ses thématiques.

{% hint style="info" %}
La gestion de l’univers d'[_ecologie_**.data.gouv**_.fr_](https://ecologie.data.gouv.frhttps/ecologie.data.gouv.fr/) évolue vers une sélection jeu de données par jeu de données. Cette évolution prévue pour le premier trimestre 2026 permettra le référencement ciblé de jeux de données en provenance d'organisations dont le champ d’action dépasse les seules thématiques environnementales (collectivités territoriales, associations spécialisées dans l'information géolocalisée, etc.).
{% endhint %}

Origine des données actuellement référencées sur [_ecologie_**.data.gouv**_.fr_](https://ecologie.data.gouv.frhttps/ecologie.data.gouv.fr/) :&#x20;

* administration centrale (SDES, startups d'État, etc.) ;&#x20;
* services déconcentrés de l'État (DREAL, DEAL, DRIEAT, DDT(M), DTAM) ;
* opérateurs (IGN, Météo-France, Ademe, BRGM, Cerema, etc.) ;
* associations spécialisées (Airparif, Agence ORE, etc.) ;
* syndicats mixtes (parcs naturels, etc.) ;&#x20;
* GIP (Agence Bio, etc.) ;
* sociétés anonymes (EDF, Engie, etc.)
