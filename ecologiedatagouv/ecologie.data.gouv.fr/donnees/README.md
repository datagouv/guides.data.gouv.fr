# Données

Le catalogue [_ecologie_.**data.gouv**._fr_](../../) offre un accès centralisé à des données actualisées produites par des acteurs de référence.

Près de **30 000 jeux de données** sont disponibles sur le catalogue [_ecologie_.**data.gouv**._fr_](../../).&#x20;

Cette collection constitue une **base évolutive**, appelée à s'enrichir et à s'affiner grâce aux besoins exprimés par les acteurs partenaires.
