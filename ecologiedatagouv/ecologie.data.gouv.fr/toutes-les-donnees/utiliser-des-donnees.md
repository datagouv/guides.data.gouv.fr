# Utiliser des données

Les jeux de données sont décrits par des fiches qui leur sont propres, les informations qu'elles contiennent sont décrites ci-dessous.&#x20;

### Comprendre un jeu de données

#### Le bandeau de description

<figure><img src="../../.gitbook/assets/image (40).png" alt=""><figcaption><p>Jeu de données "DPE Logements existants" (depuis juillet 2021) fourni par l'Ademe</p></figcaption></figure>

Le bandeau de description permet d'identifier les informations clés concernant un jeu de données :&#x20;

* son **titre** ;&#x20;
* sa **description,** telle que renseignée par l'organisation source. On peut y retrouver le contexte de production du jeu de données et d'éventuels liens utiles ;&#x20;
* l'**organisation** source ainsi que la date de **dernière mise à jour** du jeu de donnée ;&#x20;
* la **licence** associée au jeu de donnée ;
* le score de qualité des **métadonnées** qui permet de distinguer les jeux de données de qualité.
  * <a href="https://guides.data.gouv.fr/guides-open-data/guide-qualite/ameliorer-la-qualite-dun-jeu-de-donnees-en-continu/ameliorer-le-score-de-qualite-des-metadonnees" class="button secondary">En savoir plus sur le score de qualité des métadonnées.</a>

{% hint style="success" %}
L'amélioration des métadonnées des jeux de données est un processus constant. Nous sollicitons nos contributeurs pour participer à cette amélioration.
{% endhint %}

#### L'onglet "Informations"

Cet onglet détaille la **fiche de métadonnée** du jeu de donnée en précisant les conditions d'accès et d'utilisation, la licence, la généalogie, la date de création, la fréquence de mise à jour et la date de la dernière mise à jour pour chaque jeu de données. Elle mentionne aussi la couverture territoriale du jeu de données.&#x20;

### Accéder aux données

#### L'onglet "Fichiers"&#x20;

<figure><img src="../../.gitbook/assets/image (41).png" alt=""><figcaption><p>Jeu de données "Aires géographiques des AOC/AOP" fourni par l'INAO</p></figcaption></figure>

L'accès aux données se fait dans l'onglet "Fichiers", où les données peuvent être :&#x20;

* **téléchargées en fichier CSV (**_**comma separated values**_**) :** Il s'agit du format le plus utilisé pour échanger des données. Les outils bureautiques de type tableur (Microsoft Excel, LibreOffice Calc) permettent de lire ces fichiers et de les afficher de manière intelligibles à condition de sélectionner le séparateur "virgule" dans la fenêtre d'import du fichier. Les outils d'édition de texte permettent aussi d'ouvrir ces fichiers, notamment pour trouver la cause d'une erreur lors de l'import dans un tableur.
* **visualisées grâce à l'outil** [_**explore**_**.data.gouv.**_**fr**_](https://explore.data.gouv.fr) **:** L'outil permet d'obtenir un aperçu des données, en savoir plus sur les différentes colonnes ou réaliser des filtres et des tris.

{% hint style="warning" %}
A ce stade, seuls les fichiers tabulaires de moins de 100Mo sont visualisables sur _explore_.**data.gouv.**_fr._
{% endhint %}
