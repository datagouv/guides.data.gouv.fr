# Publier des données

{% stepper %}
{% step %}
### Créer un compte utilisateur et rejoindre une organisation&#x20;

La documentation [**data.gouv**._fr_](https://data.gouv.fr) vous offre un accompagnement pas à pas dans la création de votre compte utilisateur et vous explique comment rejoindre une organisation.&#x20;

Tel que mentionné ci-dessus, votre compte utilisateur est le même pour [_ecologie_.**data.gouv**._fr_](../../) et [**data.gouv**._fr_](https://data.gouv.fr/)_._&#x20;

{% embed url="https://guides.data.gouv.fr/compte-utilisateur/creer-un-compte-utilisateur" %}
{% endstep %}

{% step %}
### Publier vos données&#x20;

1. **Rendez-vous sur :** [**data.gouv.**_**fr**_](http://data.gouv.fr)
2. **Choisir le mode de mise en ligne.**&#x20;

Plusieurs modes de mises en ligne sont possibles sur la plateforme [data.gouv._fr_](https://data.gouv.fr) :&#x20;

* **Publication manuelle sur** [**data.gouv.**_**fr**_](https://data.gouv.fr) **;**
* **Par API ;**
* **Par moissonnage.**

Nous vous renvoyons à la documentation [data.gouv._fr_](https://data.gouv.fr) pour un accompagnement pas à pas dans la publication de vos données.&#x20;

{% embed url="https://guides.data.gouv.fr/jeux-de-donnees/publier-un-jeu-de-donnees" %}
{% endstep %}

{% step %}
### Déclarer votre organisation dans l'univers ecologie.data.gouv.fr

Pour que votre jeu de données soit visible sur [_ecologie_.**data.gouv**._fr_](http://ecologie.data.gouv.fr), veuillez vérifier que votre organisation soit déclarée dans le portail. Si ce n’est pas le cas, contactez l'équipe en charge du catalogue : [ecospheres@developpement-durable.gouv.fr](mailto:ecosphreres@developpement-durable.gouv.fr).&#x20;
{% endstep %}
{% endstepper %}

En cas de difficulté, contactez [ecospheres@developpement-durable.gouv.fr](mailto:ecosphreres@developpement-durable.gouv.fr). Nous accompagnons les gestionnaires de catalogues du pôle ministériel dans ces démarches.&#x20;
