# Comprendre l'éligibilité à l'espace agent

Tous les agents des administrations au sens de [l'article L100-3](https://www.legifrance.gouv.fr/codes/article_lc/LEGIARTI000031367308) du Code des Relations entre le Public et l'Administration peuvent se connecter et utiliser l’espace agent public. En se connectant, ils s'engagent à respecter les [modalités d'utilisation du service](https://annuaire-entreprises.data.gouv.fr/modalites-utilisation).

La [liste des administrations L100-3](https://www.data.gouv.fr/fr/datasets/liste-des-administrations-francaises/) est consultable sur data.gouv.fr. Elle est publiée par la DINUM et elle est régulièrement mise à jour.

{% hint style="warning" %}
Attention, la liste sur data.gouv.fr inclut aussi les administrations qui ne sont pas L100-3. Une colonne particulière permet de distinguer les L100-3 des autres et savoir si une administration a accès à l'espace agent.
{% endhint %}
