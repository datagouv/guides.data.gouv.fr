# Plafond de consultation de données

Pour des raisons de sécurité, nous avons mis en place un nombre de\
consultation de données maximum autorisé sur une plage horaire. Toutes les sources de données de l’espace agent sont concernées.



Les plafonds sont les suivants :&#x20;

| Fenêtre temporelle (glissante) | Nombre de consultations autorisées |
| ------------------------------ | ---------------------------------- |
| 10 minutes                     | 100                                |
| 1 heure                        | 200                                |
| 1 jour                         | 1 000                              |
| 1 semaine                      | 5 000                              |



### Que faire si j’ai atteint le plafond ?

Si vous avez atteint le plafond, nous en avons été notifiés. Il n’y a rien d’autre à faire qu'attendre que la plage horaire soit totalement écoulée.&#x20;



Si les plafonds sont insuffisants pour votre usage, vous pouvez [nous contacter](../echanger-avec-notre-equipe/nous-ecrire.md) pour que nous en discutions.
