# Types de données et habilitations

### Données open data

La liste des données open data est disponible sur le site [https://annuaire-entreprises.data.gouv.fr/donnees/sources](https://annuaire-entreprises.data.gouv.fr/donnees/sources).&#x20;

Le site grand public de l’Annuaire des Entreprises rassemble les principales données publiques sur les entreprises, administrations et associations de France (RNE, Sirene, RNA sont les principales sources) et les structure en un format cohérent.&#x20;

### Données non-open data

L'espace agent permet aux agents publics de consulter des données qui ne sont pas open data. Son accès est réservé à l’administration.

Attention, toutes ces données n’ont pas le même niveau de sensibilité. Nous classons ces données en deux catégories :&#x20;

* Les **données accessibles pour tous les agents**. Le fait d’appartenir à une administration suffit à accéder aux données
* Les **donnes accessibles uniquement aux agents qui travaillent dans un cadre juridique** **précis**
