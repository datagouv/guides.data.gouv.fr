# Administrer votre groupe de travail

## Rôle d'administrateur

Lorsque vous obtenez une habilitation pour accéder à des données supplémentaires, l’onglet "Mes Groupes" s’active dans votre espace agent.

En tant qu’administrateur d’un groupe, vous pouvez :

* Renommer le groupe
* Ajouter un membre
* Modifier le rôle d'un membre (utilisateur ou administrateur)
* Supprimer un membre
* Mettre à jour l'habilitation liée à votre groupe si votre cadre juridique évolue

Tous les membres d’un groupe partagent les mêmes droits d’accès aux données supplémentaires. Un groupe peut avoir plusieurs administrateurs.&#x20;

{% hint style="warning" %}
L’administrateur est **légalement responsable** de donner ces droits uniquement à des personnes :

* exerçant dans le même cadre juridique que lui
* travaillant pour l’organisation pour laquelle il est habilité
* répondant à [nos conditions d'utilisation](https://annuaire-entreprises.data.gouv.fr/modalites-utilisation)
{% endhint %}
